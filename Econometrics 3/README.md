# Econometrics III

**Institution:** NYU Stern, M.S. Quantitative Economics  
**Topics:** Machine learning methods, dimensionality reduction, gradient-based optimization

## Notebooks

| Notebook | Description |
|----------|-------------|
| `Neural networks.ipynb` | Feedforward neural network implemented from scratch in NumPy — forward pass, backpropagation, gradient descent |
| `PCA.ipynb` | Principal component analysis via eigendecomposition and SVD |
| `Functional optimization with line search.ipynb` | Gradient descent with Armijo and Wolfe line search conditions |
| `Regression with line search.ipynb` | Linear regression solved via gradient descent with line search |
