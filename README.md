# MSQE — NYU Stern

Coursework from MS Quantitative Economics (MSQE) at NYU Stern School of Business. Each folder contains implementations and assignments from a specific course, covering methods in econometrics, optimization, machine learning, and economic theory.

## Courses

| Folder | Course | Key Methods |
|--------|--------|-------------|
| `Computation 2` | Computation II | Linear programming, Poisson regression with two-way fixed effects |
| `Econometrics 2` | Econometrics II | Maximum likelihood estimation via Newton-Raphson and BHHH |
| `Econometrics 3` | Econometrics III | Neural networks, PCA, gradient-based optimization with line search |
| `Empirical Industrial Organization` | Empirical IO | BLP demand estimation, nested fixed point algorithm, numerical integration |
| `Macroeconomics 1` | Macroeconomics I | Dynamic general equilibrium |
| `Macroeconomics 2` | Macroeconomics II | Dynamic stochastic general equilibrium, time-series filtering methods |

## Language and Tools

Python (primary). Key libraries: `numpy`, `pandas`, `scipy`, `pyblp`, `matplotlib`.

## Notes

Implementations from coursework completed 2024–2025. Some notebooks may require specific data files or package versions to run.
